Pomodoro Timer Technique has six steps

  1. Decide on the task to be done.
  2. Click Start (this will start a 25 minute timer).
  3. Work on the task till timer goes off.
  4. End work when the timer rings and click the Short Break button
    and click start (this will start a 5 minute timer).
  5. Repeat this four times.
  6. Then press the Long Break button and click start (this will start a 10 minute timer).
